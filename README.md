# Прикорм-трекер (PWA)

## Запуск
1) Установи Node.js 18+ (лучше 20)
2) В папке проекта:
   npm i
   npm run dev

Открой: http://localhost:3000

## PWA
В продакшене (после build) приложение можно "Установить" на телефон/ПК.

## Редактирование плана
На вкладке **План** есть кнопка **Редактировать**:
- можно менять текст плана для каждого времени
- можно менять "Фокус" дня/недели
- правки сохраняются офлайн (IndexedDB)
- есть экспорт/импорт правок (JSON) для переноса на другое устройство

## Деплой через GitHub + установка на iPhone (рекомендовано)
PWA на iPhone требует **HTTPS**, поэтому самый простой путь — GitHub (код) + Vercel (хостинг).

### 1) Залить проект в GitHub
1. Создай репозиторий на GitHub (например `prikorm-pwa`).
2. В папке проекта:
   - `git init`
   - `git add .`
   - `git commit -m "init"`
   - `git branch -M main`
   - `git remote add origin <URL_репозитория>`
   - `git push -u origin main`

### 2) Подключить Vercel к репозиторию
1. Зарегистрируйся на Vercel.
2. New Project → Import Git Repository → выбери `prikorm-pwa`.
3. Framework: Next.js (Vercel определит автоматически).
4. Deploy.

### 3) Установить на iPhone
1. Открой в **Safari** ссылку на задеплоенный сайт (vercel.app домен).
2. Нажми **Поделиться** (квадрат со стрелкой вверх).
3. Выбери **На экран "Домой"**.
4. Готово — откроется как приложение.

> Важно: ставить лучше именно из Safari (так надежнее для PWA на iOS).
